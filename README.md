# 🏎️ F1 SCALE MODEL ARCHIVE: LEYENDAS A ESCALA

> **Catálogo interactivo de los 15 vehículos más históricos de la Fórmula 1, presentados a través de réplicas de colección.**

---

## 📂 01. DESCRIPCIÓN DEL PROYECTO

Este proyecto es una **vitrina digital** desarrollada para la materia Aplicaciones Web Cliente. El objetivo es ofrecer una experiencia visual y técnica sobre la evolución de la Fórmula 1, utilizando como base modelos a escala de alta fidelidad.

El catálogo permite explorar la ingeniería de los monoplazas, desde la simplicidad mecánica de los años 50 hasta la complejidad aerodinámica de la actualidad, destacando hitos como el **Lotus 79** o el **McLaren MP4/4**.

---

## 🛠️ 02. STACK TECNOLÓGICO

Para el desarrollo de esta plataforma se utilizaron estándares modernos de maquetación, enfocados en el rendimiento y la organización del código:

### 🔹 Maquetación y Estructura (HTML5)

- Uso de **etiquetas semánticas** (`<article>`, `<section>`, `<nav>`) para una arquitectura limpia.
- Implementación de atributos de datos (`data-attributes`) para facilitar el manejo de información desde el script.

### 🔹 Estilos y Diseño (CSS3)

- **Grid Layout & Flexbox:** Para una grilla de exposición responsiva que se adapta a cualquier dispositivo.
- **Custom Properties (Variables):** Gestión centralizada de colores institucionales de las escuderías.
- **UI/UX:** Diseño con estética de "Garaje Premium", utilizando fondos oscuros y tipografías técnicas.

### 🔹 Lógica de Interacción (JavaScript)

- Filtrado dinámico de vehículos por **Era** (Classic, Aero, Modern).
- Manipulación del DOM para la carga eficiente de las fichas técnicas.
